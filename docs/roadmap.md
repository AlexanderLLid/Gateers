# Roadmap — Wants to Have

Things we've decided we want but are deferring to a later phase. These are
*intended*, not undecided — genuinely unsettled questions live in open-questions.md.
When an item comes into scope, build its pages and strike it here.

## World / places
- **Orbital / space Gates** — free-floating Gates with no planet beneath; cannot be
  turtled, so always-live PvP zones. (The greybox proves ground Gates first.)
- **Megastructure / Supergate sites** — rare precursor megastructures that pass
  fleets; server-event scale; the macro-expansion path between systems.

## Systems
- **Space / fleet combat** — the two-phase structure: fleet raids for orbital
  control, then infantry raids on the base. Ground/infantry combat ships first.
